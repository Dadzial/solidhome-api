# SolidHomeApi

Backend API project for a Smart Home system.

## Overview
This is a backend service designed to support an **Angular** frontend application. The system is integrated with an **NXP board** to provide smart home functionalities.

## Tech Stack
- **Backend:** Node.js (Express/TypeScript)
- **Frontend:** Angular
- **Hardware:** NXP Board

## Api is in development
The API is currently under development, and we are working on implementing the necessary endpoints to support the frontend application and hardware integration.
